# LobsterMax

A clean, OpenClaw-only “maxing” guide: short patterns for prompts, tools, skills, and safety.

## Local dev
Open `web/index.html` in a browser.

## Publish
This repo is set up for GitHub Pages style publishing using the `/docs` folder.

- Edit files under `/web`
- Copy `/web` → `/docs` before pushing

(Automate later with a script or GitHub Action.)
